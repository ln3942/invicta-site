# INVICTA public spine (GitHub Pages)

This repository is a **zero-cost** static site intended for GitHub Pages.

It contains:
- A simple landing page (`index.html`)
- A **Gold-standard PIRTOE WFF / Substring Template** block
- Reference images (dashboard, peer review, schema snapshots)

## Deploy (GitHub Pages)
1. Upload these files to your repo root (or replace existing site files).
2. In GitHub: **Settings → Pages → Deploy from branch** (e.g., `main` / `/root`)
3. Visit the Pages URL.

## Custom domain (optional)
If you want `invictaresearch.ai`, keep the `CNAME` file and set your DNS:
- Add `CNAME` record for `www` to your GitHub Pages domain
- Add `A` records for apex domain per GitHub guidance

No paid services required.
